My Skills and Blueprints

01. HTML & Structure

HTML is the skeleton of your project. It defines the hierarchy, from the main title to the smallest text elements. Good structure is essential for high-quality, accessible websites.

I use tags like <section>, <div>, and <header> to keep everything organized and easy to read.

02. CSS & Design

CSS (like Tailwind) provides the style and color. This is where the black background and vibrant accents come from, making the design modern and visually striking.

Responsive design ensures projects look great on mobile phones, tablets, and desktop screens—crucial for a modern web experience.

03. App Development

This involves adding actions and logic using visual tools like **MIT App Inventor**. Block-based programming allows me to build functions, like buttons that respond to clicks and manage the game state of my apps.

Focusing on smooth behavior and interaction design makes the app feel polished and professional for any user.

High-quality web design isn't just about looking good; it's about making the experience intuitive and fast for the user. I focus on clean code and performance so that my creations load instantly and work flawlessly, no matter the device.

To truly master web development, practicing consistently and breaking down large problems into smaller, manageable chunks is key. Every element on this page—from the navigation bar to the section dividers—is a small, self-contained HTML component styled by Tailwind CSS.

Clicker Game

**Platform:** MIT App Inventor

This project demonstrates event-driven programming, game loop logic, and visual UI design principles using block-based coding. In this game a baseball is trying to get across the field for a home run and the player must click on it to get points. Certain things the player does may lose them points and it is all about letting in the least amounts of runs possible.

Detailed Project Breakdown:

This game has different features to it that differentiate it from other games that may seem similar in mechnaics to it. For one, this game uses a small animation like feature that changs the deisgn of the sprite when clicked to not only show a point was scored for the player but to also add a little more depth to the enique game. It also has a baseball field that faces from home plate on the left to the outfild on the right to simulate runs being scored by the opposing team, the computer. It also has a feature to where if the computer picks a certain number the sprite will speed up or slow down to make the game more exciting. It lasts 20 second with the ability to restart instantly. This game uses unique features and ideas to make it a fun and engaging clicker game for all the baseball fans out there.**
